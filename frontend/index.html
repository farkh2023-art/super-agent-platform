💡 Configuration via .env
# Copiez .env.example en .env
AI_PROVIDER=mock
ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
OLLAMA_BASE_URL=http://localhost:11434

Les clés API sont chargées depuis les variables d'environnement pour la sécurité. Elles ne sont jamais retournées par l'API.
