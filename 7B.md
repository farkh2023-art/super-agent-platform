2\. ALERT RULES

==================================================



Créer :



backend/src/alerts/alertRules.js

backend/src/alerts/alertEvaluator.js

backend/src/routes/alerts.js



Modèle règle :

\- id

\- name

\- enabled

\- severity: info|warning|critical

\- component: system|auth|storage|rag|scheduler|agents

\- metric

\- operator: gt|gte|lt|lte|eq|neq

\- threshold

\- cooldownMs

\- createdAt

\- updatedAt



Routes :

GET /api/alerts/rules

POST /api/alerts/rules

PUT /api/alerts/rules/:id

DELETE /api/alerts/rules/:id

POST /api/alerts/evaluate



Règles par défaut :

\- memory usage > 85%

\- storage desync alerts > 0

\- auth failed login rate > threshold

\- scheduler failed jobs > 0

\- RAG embeddings disabled warning si MEMORY\_EMBEDDINGS=true mais unavailable
