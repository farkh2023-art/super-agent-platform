# User Guide

Run `.\release\install.ps1`, then `.\release\start.ps1 -Mode demo`, and open `http://localhost:3001`.

Use the dashboard as the command center. Create tasks, inspect agents, build workflows, add memory context, review Alert Center, export Admin Health reports and create backups from Settings or `.\release\backup.ps1`.

Demo mode requires no API key. Single mode is for local use. Multi mode enables users, workspaces, sessions and audit logs.
